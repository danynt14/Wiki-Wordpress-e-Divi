Parallax e
Overlay

Quando o usuário adiciona um imagem na sessão e marca a opção de parallax, assim que ele adiciona um gradiente sobre a sessão["Overlay"], o efeito de parallax não acompanha o gradiente, perdendo a compatibilidade com a imagem que foi adicinada, diminuindo a senssação do efeito. Para isso é preciso adicionar o código abaixo na sessão de confirguração global no DIVI: e adicionar a classe css gradient-overlay na sessão onde foi inserido a imagem.

.gradient-overlay:before {
content:"";
position:absolute;
z-index:1;
background:inherit;
top:0;
bottom:0;
width:100%;
}
.gradient-overlay .et_pb_row {
z-index:2 !important;
}
.gradient-overlay {
background-color: transparent;
}
